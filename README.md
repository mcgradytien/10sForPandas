#### 创建对象
可以参考 Data Structure Intro section (http://pandas.pydata.org/pandas-docs/stable/dsintro.html#dsintro)

通过传递一组列表数据来创建 Series对象，pandas会创建默认的整数的索引
```
In [4]: s = pd.Series([1,3,5,np.nan,6,8])

In [5]: s
Out[5]: 
0    1.0
1    3.0
2    5.0
3    NaN
4    6.0
5    8.0
dtype: float64
```
